# Meta tags to add to each page

For every file below, find this in the `<head>`:
```
<title>...</title>
```
and add these lines right after it (before `<style>` or `</head>`), then also add the favicon line once per page:
```
<link rel="icon" type="image/svg+xml" href="favicon.svg">
```

---

```html
<meta name="description" content="Simple, elegant, premium recipes crafted for the home cook. Browse dinner, breakfast, dessert, healthy, and quick-meal recipes.">

<meta property="og:description" content="Simple, elegant, premium recipes crafted for the home cook.">
```
